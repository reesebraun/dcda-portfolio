Semester Reflection

What I learned: I learned how to set up a personal portfolio using GitHub, VS Code, and GitHub Pages, and how these tools work together to publish a website. I also became more comfortable editing HTML and understanding how small changes in code affect the structure and content of a webpage.

Challenges I overcame: One challenge I faced was uploading my profile photo and adjusting its size so it fit properly within the page layout, but through troubleshooting file paths and experimenting with CSS styling, I was able to resolve the issue and improve the overall design.

How I grew: My understanding of digital culture and data analytics has evolved as I've learned to use tools like Tableau and Folium to visualize data and tell stories with numbers. I've also become more confident in my ability to create and maintain a professional online presence.

Looking ahead: I plan to continue building on these skills by exploring more advanced data visualization techniques and learning how to integrate interactive elements into web pages.
